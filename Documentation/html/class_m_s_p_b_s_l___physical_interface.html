virtual void MSPBSL_PhysicalInterface::invokeBSL ( ) [pure virtual]

Allows each physical interface to decide how it invokes the BSL.

This function is required to do anything, as some BSLs (such as USB) are not invoked by any specific controllable action on the part of the physical interface.

Implemented in MSPBSL_PhysicalInterfaceSerialUART, and MSPBSL_PhysicalInterfaceUSB.
