uint16_t MSPBSL_Connection5xxUSB::loadRAM_BSL ( uint8_t *  password)

USB-specific helper function to load the RAM BSL and start it

This function is included in the 5xx USB class as a convenience function it's purpose is to make it simple to unlock a device, load and start the RAM BSL, and re-connect to this RAM BSL

Parameters:
passthe password to submit to the connected device's BSL
Returns:
an unsigned byte describing the success or failure of the entire operation 0: No error, RAM based BSL is loaded, and further communication can continue using this class >0: one or more errors occured
